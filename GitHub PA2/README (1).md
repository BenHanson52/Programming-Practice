## 🎯 Learning Objectives

By completing this assignment, you will:

- Model a traffic network using systems of linear equations
- Analyze flow conservation at intersections
- Interpret free variables in a physical context
- Implement and modify a max-flow style program
- Analyze how structural changes affect system behavior
